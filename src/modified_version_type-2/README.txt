Модификация ТИП-2 от Классической версии отличается тем что вместо одного алгоритма,
параллельно запускается несколько алгоритмов, с каждого получается по решению, и из получившихся решений,
выбирается лучшее. Но чтобы количество вызовов целевых функций оставалось прежним,
следует делить количество иттераций на количество машин, или количество индивидов на количество машин, 
так-как количество вызовов целевой функции задаётся формулой (количество индивидов * количество иттераций).
