Модификация ТИП-0 от Классической версии отличается лишь тем,
что между машинами распределенны выполнения целевых функций.

Это крайне эффективно в задачах с медленными целевыми функциями.


Если данные между Классической версией, Типом-0 и Типом-1 сильно отличаются, это значит что что-то
работает не так, ибо с точки зрения среднеквадратичной ошибки, алгоритмы работают одинаково.
